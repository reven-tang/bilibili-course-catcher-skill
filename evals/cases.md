## 评分验证用例

### SV1: 完整评分计算
**课程数据**:
- 内容质量：体系完整(2.0) + 时长>5h(1.5) + 30天内更新(1.5) + 有讲义(1.0) = 6.0 → 6.0/6.0×10 = 10分
- 用户口碑：点赞比8%(8) + 收藏率2%(7) + 有弹幕(1) = 8.5 → 8.5/10×10 = 8.5分
- 匹配度：年级匹配(10) + 难度匹配(10) + 类别匹配(10) = 10分
- UP主可信度：认证(3) + 月更(2) + 标题含老师(2) + 粉丝>10万(1) = 8分
- 时效性：2026版 = 10分

**预期总分**: 10×0.30 + 8.5×0.25 + 10×0.20 + 8×0.15 + 10×0.10 = 9.125分

### SV2: 无链接课程排除
**输入**: 搜索结果中有1门课程无bvid
**预期**: ❌ 该课程不进入评分，直接排除
