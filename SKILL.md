# B 站课程抓取技能

从 B 站（bilibili）智能抓取并推荐优质学习课程，通过**多轮交互**逐题收集学生需求，基于 5 维度生产级评分系统筛选最佳课程。

## 核心功能

### 1. 多轮交互式问答
**每轮只问一个问题**，逐题推进，降低用户负担。

**问答流程（4轮必问 + 2轮可选）**：

| 轮次 | 问题 | 选项 | 必问？ |
|------|------|------|--------|
| 第1轮 | 学生目前是哪个年级？ | A.小学 B.初中 C.高中 D.大学 E.其他 | ✅ 必问 |
| 第2轮 | 需要哪类课程？ | **根据年级动态调整**（见下方课程映射表） | ✅ 必问 |
| 第3轮 | 课程难度级别？ | **根据类别动态调整**：语文→6选项（+作文专项+阅读理解专项）；其他类别→4选项 | ✅ 必问 |
| 第4轮 | 更新时间偏好？ | A.最近半年 B.最近一年 C.不限（经典课程也可） | ✅ 必问 |
| 第5轮 | 是否偏好认证UP主？<br>*认证UP主经过B站官方审核，教学内容更有保障* | A.是，只看认证UP主 B.不限 | ❌ **必展示** |
| 第6轮 | 视频时长偏好？<br>*短课单集<30min碎片化，中课系统性较强，长课内容最全* | A.短课（单集<30min） B.中课（30-60min/集） C.长课（>60min/集） D.不限 | ❌ **必展示** |

**年级→课程类别映射表**（第2轮选项根据第1轮答案动态调整）

| 年级 | 可选课程类别 | 说明 |
|------|-------------|------|
| 小学 | 语文、数学、英语、编程、艺术、音乐、体育、其他 | 小学无理化生，语文是主科 |
| 初中 | 语文、数学、英语、物理、化学、生物、历史、地理、编程、体育、其他 | 初中开始接触理化生 |
| 高中 | 语文、数学、英语、物理、化学、生物、历史、地理、政治、编程、体育、其他 | 高中加政治 |
| 大学 | 编程、数学、英语、物理、其他专业课程 | 大学以专业技能为主 |
| 其他 | 语文、数学、英语、物理、化学、生物、历史、地理、编程、艺术、音乐、体育、其他 | 全类别开放 |

**类别→难度级别映射表**（第3轮选项根据第2轮答案动态调整）

| 类别 | 可选难度级别 | 说明 |
|------|-------------|------|
| 语文 | A.入门 B.标准 C.拔高 D.提优 **E.作文专项** **F.阅读理解专项** | 语文额外加两个专项训练 |
| 其他类别 | A.入门 B.标准 C.拔高 D.提优 | 保持默认4个难度 |

- **入门**：零基础，从头开始
- **标准**：课本同步，巩固基础
- **拔高**：跳出课本，思维拓展/奥数启蒙
- **提优**：竞赛冲刺，深度训练
- **作文专项**：专门针对写作能力提升（仅语文）
- **阅读理解专项**：专门针对阅读理解技巧训练（仅语文）

**快速触发**：用户消息已包含年级/类别/级别时，自动提取参数，跳过对应轮次。

**跳过机制**：用户回答"不限"或"无所谓"时，使用默认值进入下一轮。

**⚠️ 可选轮次必须展示，不能跳过**：第5轮和第6轮虽然标记为可跳过（用户可答"不限"），但**每轮都必须向用户展示问题**，由用户决定是否回答。不能因为"可选"就自动隐藏。

### 2. 生产级评分系统
**5 维度加权评分**（权重总和=1.0）—— 每个维度都有实际数据支撑，无无效权重：

| 维度 | 权重 | 数据源 | 评分标准 |
|------|------|--------|----------|
| 📚 内容质量 | 30% | 时长+集数+体系性+更新频率 | 量化评分（见下方细则） |
| 💬 用户口碑 | 25% | 点赞比+收藏率(收藏/播放)+弹幕互动 | 比率映射，消除播放量通胀 |
| 🎯 匹配度 | 20% | 年级匹配+难度匹配+类别匹配 | 与用户需求的精确匹配程度 |
| 👤 UP主可信度 | 15% | 教育认证+更新频率+专业度 | 认证+持续产出 > 粉丝数 |
| 🕐 时效性 | 10% | 课程更新时间+版本标注 | 2026版>2025版>更早 |

**评分细则**：

📚 **内容质量**（0-10分）：
- 体系完整性：系列课程(>20集)+2.0 | 单视频+0
- 时长合理性：>5h+1.5 | 2-5h+1.0 | <2h+0.5
- 更新频率：30天内更新+1.5 | 90天内+1.0 | 更早+0.5
- 配套资源：有讲义/PDF+1.0 | 无+0

💬 **用户口碑**（0-10分）：
- 点赞比：>8%→10 | 5-8%→8 | 3-5%→6 | <3%→4
- 收藏率(收藏/播放)：>3%→10 | 1-3%→7 | <1%→4
- 弹幕互动：有弹幕讨论+1 | 无+0

🎯 **匹配度**（0-10分）：
- 年级精确匹配：标题/简介包含目标年级→10 | 包含年级范围→7 | 不明确→4
- 难度匹配：入门课vs入门需求→10 | 拔高vs拔高→10 | 错配→3
- 类别匹配：精确类别→10 | 相关类别→6 | 不相关→2

👤 **UP主可信度**（0-10分）：
- 教育认证：B站教育认证→+3 | 无→+0
- 更新频率：月更→+2 | 季更→+1 | 停更→+0
- 专业度：标题含"老师"/"课堂"→+2 | 动画站→+1 | 杂类→+0
- 粉丝数>10万→+1 | <10万→+0（仅1分加成，不主导）

🕐 **时效性**（0-10分）：
- 2026版/标注→10 | 2025版→8 | 2024→6 | 更早→3

### 3. 筛选条件
- 年级：小学/初中/高中/大学/其他
- 类别：根据年级动态调整（语文/数学/英语/物理/化学/生物/历史/地理/政治/编程/艺术/音乐/体育/其他）
- 级别：根据类别动态调整（语文：入门/标准/拔高/提优/作文专项/阅读理解专项；其他：入门/标准/拔高/提优）
- 可选偏好：
  - 更新时间（最近半年/最近一年/不限）
  - 认证 UP 主
  - 视频时长（短/中/长/不限）

### 4. 输出格式
- Markdown 表格 + 详细卡片
- 金/银/铜牌标识（前 3 名）
- 综合评分 + 5 维度详细指标
- **必须包含以下列：排名、课程名称、UP主、课程链接、综合评分**
- **硬性要求：每门推荐课程必须携带完整链接**
  - 格式：`https://www.bilibili.com/video/{bvid}`
  - 无链接的课程不进入推荐列表
  - 链接不可点击时用降级格式：`bilibili.com/video/{bvid}`
- **UP主列要求**：必须从搜索结果中提取UP主名称，不可留空或用默认值

## 技术实现

### 数据获取策略
**混合模式**：
1. 优先调用 B 站官方 API
2. API 失败时降级到网页爬取
3. 使用 browser 工具绕过反爬

### 评分算法
```typescript
总分 = 
  内容质量 × 0.30 +
  用户口碑 × 0.25 +
  匹配度 × 0.20 +
  UP主可信度 × 0.15 +
  时效性 × 0.10
```

**无链接课程不进入评分**——直接排除，不浪费计算。

### 错误处理
- 指数退避重试（最多 3 次）
- API 频率限制自动降级
- 友好错误提示

## 验收标准

- ✅ 技能能正确触发并进入问答流程
- ✅ 能收集完整的用户需求
- ✅ 能从 B 站抓取到相关课程（至少 5 条结果）
- ✅ 能按评分标准计算综合评分
- ✅ 输出格式符合规范
- ✅ 推荐课程准确率 > 80%

## Gotchas（已知陷阱）

### ❌ 不应触发
| 场景 | 示例 | 原因 |
|------|------|------|
| 纯询问无学习意图 | "B站是什么" | 无课程/学习意图 |
| 指定其他平台 | "帮我找网易云课堂的课程" | 非B站平台 |
| 非课程类内容 | "B站搞笑视频推荐" | 非学习类视频 |
| 已有具体视频 | "BV1xxx这个视频怎么样" | 用户已有目标，无需推荐 |

### ⚠️ 注意
- **小学无物理/化学/生物**：第2轮选项根据年级动态调整
- **语文专项仅语文可选**：作文专项/阅读理解专项只在类别=语文时显示
- **可选轮次必展示**：第5-6轮必须向用户展示，不能自动跳过
- **无链接课程直接排除**：不进入评分，不浪费计算

## 已知限制

1. API 频率限制：50 次/分钟
2. 数据实时变化：每次查询结果可能不同
3. 只推荐免费内容：付费课程会标注但不推荐
4. 地域限制：部分内容可能有地域限制
