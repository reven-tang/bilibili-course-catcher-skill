## 🔧 技术细节

### 向后兼容性

**旧代码**（仍支持）:
```typescript
if (preferences.preferences.requireRecent) {
  // 30 天内过滤
}
```

**新代码**（推荐）:
```typescript
const recency = preferences.preferences.recency || 'any';
if (recency === '30days') {
  // 30 天内过滤
}
```

**兼容性保证**:
- 旧字段 `requireRecent` 仍可使用
- 新字段 `recency` 优先
- 自动映射：`requireRecent=true` → `recency='30days'`

### 匹配度计算

| 时间范围 | 超过阈值后匹配度 | 说明 |
|----------|----------------|------|
| 30 天内 | ×0.3 | 严格过滤 |
| 180 天内 | ×0.3 | 中等过滤 |
| 365 天内 | ×0.3 | 宽松过滤 |
| 任意 | ×1.0 | 不过滤 |
