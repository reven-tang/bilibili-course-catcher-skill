# 课程推荐输出模板

## 标题格式
```
## 📚 为您找到 **{count}** 门优质课程

> 筛选条件：{grade} · {category} · {level} · {update} · {up认证} · {duration}
```

## 表格格式

| 排名 | 课程名称 | UP主 | 课程链接 | 综合评分 |
|------|----------|------|----------|----------|
| 🥇 | {name} | {up主} | [链接]({url}) | {score}/10 |
| 🥈 | {name} | {up主} | [链接]({url}) | {score}/10 |
| 🥉 | {name} | {up主} | [链接]({url}) | {score}/10 |
| {n} | {name} | {up主} | [链接]({url}) | {score}/10 |

**硬性要求**：
- UP主列必须从搜索结果提取，不可留空
- 课程链接格式：`https://www.bilibili.com/video/{bvid}`
- 无链接课程不进入推荐列表

## 详细卡片格式

### 🥇 {课程名称}（{综合评分}/10）

| 维度 | 评分 | 说明 |
|------|------|------|
| 📚 内容质量 | {score}/10 | {说明} |
| 💬 用户口碑 | {score}/10 | {说明} |
| 🎯 匹配度 | {score}/10 | {说明} |
| 👤 UP主可信度 | {score}/10 | {说明} |
| 🕐 时效性 | {score}/10 | {说明} |

**课程亮点**：{亮点描述}

**适合人群**：{人群描述}

---

## 无结果情况

```
## ⚠️ 未找到符合条件的课程

**可能原因**：
- 筛选条件过于严格
- 该领域优质课程较少

**建议**：
- 放宽更新时间要求（选择"不限"）
- 尝试其他难度级别
- 扩大年级范围
```
